# 🚗 Clutch or Crash: A Car Trivia Quiz

Welcome to **Clutch or Crash**, the ultimate pit stop for car enthusiasts!  
This quiz isn’t just about answering questions — it’s about proving your horsepower of knowledge. From legendary classics to modern supercars, every round will test whether you’re cruising in the fast lane or stalling in traffic.

---

## 🎮 Features
- Savage yet classy result messages for every score range.
- Car‑themed design with custom cursors and aesthetic fonts.
- Responsive layout for a smooth experience across devices.
- Fun, engaging questions tailored for gearheads and speed freaks.

---

## 🛠️ Tech Stack
- **HTML5** for structure  
- **CSS3** for styling and custom cursors/fonts  
- **JavaScript (ES6)** for quiz logic and dynamic results
